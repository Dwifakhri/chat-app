# Chat App

Real time chat app using vue 3 and socket.io

## Pre-requisite

1. Node.js version >= 18.15.0
2. Package manager (bun) (https://bun.sh/)

## Project Setup

```sh
bun install
```

### Compile and Hot-Reload for Development (client)

```sh
bun run dev
```

### Compile and Minify for Production (client)

```sh
bun run build
```

### Run Server

```sh
cd backend
```

```sh
bun index.js
```
